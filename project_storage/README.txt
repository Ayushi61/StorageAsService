Pre-requisite
1) Image by name VM_q5-1.img to be present in /var/lib/libvirt/images folder of the participating VMs
2) ansible folder in project_storage to be placed in the home directory of VMs

to run and create tue cloud use the following command: 
sudo python pxpect_auto_create.py

to fill the input: use the file datamodel/user_input.json
to perform network admin: use file hooks.json
